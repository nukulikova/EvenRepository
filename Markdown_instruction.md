# Инструкция для работы с Matkdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками (*).
Например, *вот так*.

Чтобы выделить текстполужирным, необходимо обрамить его двойными звездочками (**).
Например, **вот так**.
